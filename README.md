## Quick Start

1. Clone this repository
2. Edit `config.json` with your business information
3. Open `index.html` in a browser or deploy to any web host

## Configuration

All customization is done through the `config.json` file:

- **Business Info**: Name, tagline, phone, email, address
- **Social Media**: Facebook, Instagram, Twitter links
- **About Section**: Title and description
- **Services**: List of services with names, descriptions, and icons
- **Gallery**: Images to showcase
- **Testimonials**: Customer reviews
- **Business Hours**: Operating hours for each day
